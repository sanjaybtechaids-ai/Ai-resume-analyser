# AI Resume Analyzer

AI Resume Analyzer is an AI-powered web application that helps job seekers improve their resumes and career opportunities.

## Features
- Resume Analysis
- Skill Gap Detection
- Salary Prediction
- Company Eligibility Check
- Personalized Improvement Suggestions

## Tech Stack
- Python
- Flask
- Machine Learning
- HTML
- CSS
- JavaScript

## How It Works
1. Upload your resume.
2. AI analyzes your skills and experience.
3. Get improvement suggestions.
4. View estimated salary range.
5. Check eligibility for companies.
